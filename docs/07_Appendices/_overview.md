# 07 - Appendices

## Mục đích
Tài liệu tham khảo bổ sung, bảng thuật ngữ, và so sánh chi tiết các vendor/giải pháp để hỗ trợ các phần phân tích chính.

## Nội dung triển khai

| File | Mô tả |
|------|--------|
| `glossary.md` | Bảng thuật ngữ: Text-to-SQL, RAG, Agent, LLM, Vector DB, Semantic Layer, MCP, và các khái niệm kỹ thuật/kinh doanh liên quan |
| `references.md` | Danh mục tài liệu tham khảo: papers, reports, benchmarks, documentation links |
| `vendor_comparison.md` | Bảng so sánh chi tiết các vendor/giải pháp: tính năng, giá cả, ưu/nhược điểm, phù hợp theo use case |

## Đối tượng sử dụng
- Tất cả stakeholders
- Technical evaluation team
- Procurement team
